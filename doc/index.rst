Documentation
=============

Python package **teneva_ht_jax** (**ten**\ sor **eva**\ luation with **H**\ ierarchical **T**\ ucker and **jax**) provides a very compact implementation of basic operations in the Hierarchical Tucker (HT) format, including approximation and sampling from multidimensional arrays and multivariate functions. The program code is organized within a functional paradigm and it is very easy to learn and use.

-----

Installation instructions are provided in the repository `teneva_ht_jax <https://github.com/AndreiChertkov/teneva_ht_jax>`_.

-----

Current version "0.1.1".
